# Mushroom Body Compression Study

## Overview

This project explores whether the fruit fly mushroom body shows signs of a compression-like network structure.

The mushroom body is a brain region involved in learning and memory. Using public fruit fly connectome data, this project analyzes how neurons are connected and whether many inputs appear to be compressed into a smaller internal representation.

## Goal

The main goal is to practice applying data analysis and network science techniques to a real biological dataset.

Main question:

> Does the mushroom body show structural evidence of biological compression?

## Tools

- Python
- pandas
- SQLite
- NetworkX
- matplotlib / Plotly
- Jupyter Notebook

## Planned Work

- Load and explore connectome data
- Filter for mushroom body-related neurons
- Build a network graph from neuron connections
- Analyze connectivity patterns
- Visualize important network features
- Summarize findings in a short report
